# حالة العمل الحالية

## 1. حالة Git الحالية

- كانت شجرة العمل نظيفة قبل بدء المراجعة النهائية.
- الملف المعدل أثناء المراجعة: `assets/css/index.css` لإبقاء لون الرد السري أحمر في الوضع الداكن.
- الملفات الجديدة أثناء المراجعة: `docs/current-work-status.md` و`docs/post-movie-commands.md`.
- لا توجد ملفات معرفة أو صور أو ملفات خام غير متتبعة في حالة Git.
- لا يظهر `knowledge-import/` أو `tmp/` أو `.wrangler/` أو `.env` أو ملفات PDF/Word خام في حالة Git.
- لا يجب إضافة أي ملف خام أو سري إن ظهر لاحقًا، وبالأخص `knowledge-import/` و`tmp/` و`.wrangler/` و`.env`.

## 2. حزمة التقويم الدراسي

- صورة التقويم موجودة ومتتبعة في Git: `assets/knowledge-images/academic-calendar-1448-1449-2026-2027.jpg`.
- الصورة JPEG سليمة بأبعاد 1626×2275 بكسل.
- `src/index.js` يضيف `images` عند مطابقة نية التقويم الدراسي، بالمسار المحلي الصحيح.
- `assets/js/ai-assistant.js` يتحقق من مسار الصورة ويعرضها باستخدام DOM APIs آمنة، مع النص البديل والتعليق ورابط فتح الصورة.
- `assets/css/index.css` يحتوي بطاقة صورة متجاوبة للوضعين الفاتح والداكن.
- يلزم اختبار الموقع الحقيقي للتأكد من استجابة `/api/chat` وفتح رابط الصورة من البيئة المنشورة.

## 3. حزمة الرد المخصص

- الرمز `07/07/2027MN` يعمل قبل RAG، مع قبول اختلاف حالة الحرفين `MN` والمسافات الطرفية فقط.
- الرد هو: `نورة احلى حاجة بالحياة 🖤🖤`، وموسوم بالنوع `nora-secret`.
- كلمتا `نورة` و`نوره` ليستا شرطًا مباشرًا، وتتابعان إلى مسار المساعد الطبيعي.
- الرد السري لا يسجل كسؤال غير مجاب ولا يرجع صورًا.
- الواجهة تستخدم `textContent` وقائمة سماح للنوع، وتضيف `ai-message-nora-secret` للرد المخصص فقط.
- تم تثبيت اللون الأحمر والوزن العريض في الوضعين الفاتح والداكن.

## 4. حزمة الجوال والقائمة

- عنوان المنصة يستخدم `white-space: nowrap` وحجم خط متجاوب على الجوال.
- القائمة الجانبية مثبتة إلى اليمين وتغلق باتجاه اليمين عبر `translateX(105%)`.
- زر الثيم داخل القائمة يعيد استخدام `toggleDark()` وحالة الثيم المحفوظة نفسها.
- التاريخ والوقت داخل القائمة يعيدان استخدام `updateDateTime()` وبنفس تنسيق الشريط العلوي.
- يلزم تحقق بصري نهائي على 390×844 وعلى سطح المكتب في الوضعين الفاتح والداكن.
