# مراجعة الخدمات والموارد الخارجية

آخر مراجعة: 19 يوليو 2026.

## نطاق المراجعة

يغطي هذا الملف التكاملات الظاهرة في HTML وJavaScript فقط. لم تُراجع لوحات الخدمات أو قواعد Firebase أو قيم البيئة، ولم تُغيّر أي إعدادات أو مفاتيح أو bindings.

## Cloudflare

- `src/index.js` هو مدخل Worker.
- ملفات الواجهة تُخدم عبر Static Assets.
- المساعد يعتمد على Workers AI وVectorize.
- الأسئلة غير المجابة تعتمد على D1 عند توفر binding.
- الخادم المحلي `local-server.cjs` ثابت فقط ولا يمثل هذه الخدمات.

لا يجب تشغيل deploy أو migration أو Vectorize upsert ضمن فحوص الجودة المحلية.

## Firebase

### الاستخدام الحالي

- الصفحة الأرشيفية `10.html` تعرض عداد زيارات وتحمّل Firebase من `www.gstatic.com`.
- عند فتح الصفحة، ينشئ العداد اشتراك قراءة واحدًا عبر `onSnapshot`.
- يحاول العداد زيادة القيمة مرة واحدة لكل جلسة متصفح عبر `sessionStorage` ومعاملة Firestore.
- `assets/js/visitor-counter.js` يحتوي تنفيذًا منفصلًا للعداد، لكنه غير مرتبط حاليًا بأي صفحة HTML؛ لذلك لا يحمّل Firebase ولا ينفذ طلبات عند استخدام الصفحات النشطة.

### معالجة الأعطال

- تعذر تحميل Firebase أو قراءة Firestore يجب ألا يمنع بقية الصفحة من العمل.
- عند فشل القراءة تبقى قيمة fallback ظاهرة.
- أخطاء الخدمة لا تُعرض للمستخدم ولا تُسجل بتفاصيل قد تحتوي metadata غير لازمة.

### الأمان

- إعداد عميل Firebase موجود في كود المتصفح بطبيعته ولا يُعامل كسر خادم.
- الحماية الفعلية للقراءة والكتابة تعتمد على Firebase Security Rules وقيود المفتاح في لوحة الخدمة.
- لم تُراجع القواعد أو القيود من هذا المستودع، ولم تُغيّر في هذه المرحلة.
- لا ينبغي إضافة بيانات شخصية أو معرفات مستخدمين إلى مستند العداد.

## موارد CDN

| المورد | الاستخدام |
|---|---|
| Google Fonts | تحميل الخطوط العربية في الصفحات |
| CDNJS | Font Awesome في الصفحة الرئيسية |
| jsDelivr | مكتبات محلية الاستخدام في صانع التقارير وتحليل النتائج |
| gstatic | وحدات Firebase في الصفحة الأرشيفية |

المكتبات الخارجية المثبتة في الصفحات محددة بإصدارات. لا تُحدّث الروابط أو الإصدارات دون اختبار وظيفي وبصري مستقل.

## روابط الخدمات التعليمية

تحتوي المنصة على روابط لوزارة التعليم ومنصات تعليمية وخدمات Google Drive وغيرها. هذه الروابط:

- لا تُحمّل محتواها عند بدء الصفحة؛ الطلب يحدث عند اختيار المستخدم للرابط.
- تفتح في تبويب جديد مع `rel="noopener noreferrer"` حيث تكون روابط خارجية قابلة للنقر.
- تُراجع دوريًا بأداة الروابط المحلية وباختبار يدوي، لأن الفاحص المحلي لا يضمن استمرار المورد الخارجي.

## قرارات مؤجلة

- تقرير ما إذا كان العداد الأرشيفي ما زال مطلوبًا قبل توحيد أو إزالة التنفيذ غير المرتبط في `assets/js/visitor-counter.js`.
- مراجعة Firebase Security Rules وقيود المفتاح من لوحة Firebase بواسطة مالك الخدمة.
- إضافة مراقبة خارجية للروابط الرسمية إذا اعتُمدت خدمة مناسبة لاحقًا.
