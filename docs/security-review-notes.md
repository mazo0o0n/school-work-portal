# ملاحظات مراجعة الأمان

هذه جداول عمل للمراجعة اليدوية. الحالة الافتراضية **يحتاج تحقق** ولا تعني أن النظام آمن أو غير آمن.

## Git والأسرار

| الاختبار | المتوقع | الحالة ✅/❌/⚠️ | الملاحظات |
|---|---|---|---|
| مراجعة الملفات المرحّلة | لا توجد ملفات غير مقصودة أو حساسة | ⚠️ | يحتاج تحقق |
| البحث عن مؤشرات الأسرار | لا توجد أسرار متتبعة | ⚠️ | يحتاج تحقق دون عرض المحتوى |
