تصنيف الخطر: **منخفض**، **متوسط**، **عالي**. الحالة الافتراضية «يحتاج تحقق» ولا تعني نجاح الفحص.

## Git والأسرار
| البند | الخطر | المتوقع | الحالة ✅/❌/⚠️ | الملاحظات |
|---|---|---|---|---|
| الملفات المرحّلة | عالي | لا توجد ملفات حساسة أو غير مقصودة | ⚠️ | يحتاج تحقق |
