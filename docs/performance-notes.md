# ملاحظات الأداء

هذا تقرير اقتراحات فقط؛ لم تُضغط أو تُحوّل أو تُحذف أي صورة ولم تتغير روابطها.

## منهجية الفحص المقترحة

1. تشغيل الموقع محليًا في نافذة خاصة وعلى اتصال محاكى متوسط.
2. قياس Lighthouse للجوال وسطح المكتب أكثر من مرة.
3. مراجعة Network لتحديد الملفات الأكبر والطلبات المتكررة.
4. مراجعة Core Web Vitals وConsole دون تعطيل cache في القياس النهائي.
5. اختبار 390×844 والتفاعل مع القوائم والمساعد والأدوات.

## Checklist

- [ ] مراجعة الصور الكبيرة وأبعاد عرضها الفعلية.
- [ ] استخدام `loading="lazy"` للصور غير الظاهرة أول الصفحة عند ملاءمته.
- [ ] مراجعة حجم CSS وJavaScript والملفات غير المستخدمة.
- [ ] التأكد من عدم وجود Console warnings مؤثرة.
- [ ] تشغيل Lighthouse للجوال وسطح المكتب.
- [ ] اختبار تجربة الجوال والشبكات البطيئة.
