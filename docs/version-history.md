# سجل الإصدارات

## v1.5.1 — قيد التجهيز محليًا

- تعزيز أمان `1.html` و`3.html` ضد حقن HTML، وفرض حدود عملية على الصور وملفات Excel وCSV والصفوف والأعمدة.
- تشديد التحقق من طلبات `/api/chat` ومسارات الإدارة، وإضافة رؤوس الحماية ومنع التخزين المؤقت والتسريبات التشخيصية.
- تجهيز Rate Limiting مستقل للمحادثة ومحاولات المصادقة الإدارية، مع اشتراط `RATE_LIMIT_SALT` وقت الإصدار.
- اعتماد معرّفات Vectorize ثابتة مرتبطة بالمصدر والمحتوى، وإكمال الاستبدال الآمن للمعرّفات القديمة مع manifest قابل للتدقيق.
- إزالة بيانات تواصل شخصية ومسارات fallback قديمة وكود واجهة مؤكد عدم استخدامه.
- تصحيح إحصاء لوحة المعرفة وإضافة مهلة آمنة لطلبات المساعد.
- تحسين RTL والاستجابة للجوال وإدارة تركيز النوافذ والقائمة الجانبية وإمكانية الوصول.
- استكمال canonical وخريطة الموقع وفحص الروابط العامة، وتوسيع اختبارات الأمان والتدفقات والمرئيات.
- توحيد اتساق الإصدار بين الحزمة والواجهة والوثائق.
- الإصدار المنشور الحالي أثناء تجهيزها هو `v1.5.0`.
