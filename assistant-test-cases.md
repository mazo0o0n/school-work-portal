# اختبارات مساعد المنصة

## 1. أسئلة تعريف المنصة بصيغ مختلفة

| السؤال | النتيجة المتوقعة | المعلومة المتوقعة باختصار |
|---|---|---|
| ما هي منصة التنظيم المدرسي؟ | يجيب | المنصة بوابة رقمية تجمع الأدلة والأنظمة والمنصات والأدوات المدرسية. |
| وش منصة التنظيم المدرسي؟ | يجيب | يشرح أنها بوابة لتنظيم موارد المدرسة في مكان واحد. |
| ايش منصة التنظيم المدرسي؟ | يجيب | يوضح فكرة المنصة وفائدتها للإدارة والمعلمين. |
| عرفني على المنصة | يجيب | يقدم تعريفًا مختصرًا للمنصة. |
| وش فايدة المنصة؟ | يجيب | تقليل تشتت الروابط والملفات وتسهيل الوصول للموارد. |

## 2. أسئلة الأدوات

| السؤال | النتيجة المتوقعة | المعلومة المتوقعة باختصار |
|---|---|---|
| وش صانع التقارير؟ | يجيب | أداة لإنشاء تقارير تشغيلية وتربوية. |
| وين ألقى صانع التقارير؟ | يجيب | موجود في ركن الدعم المدرسي ورابطه `1.html`. |
| رابط صانع التقارير | يجيب | الرابط الداخلي `1.html`. |
| كيف أسوي تقرير؟ | يجيب | باستخدام صانع التقارير في `1.html`. |
| وش تحليل النتائج؟ | يجيب | أداة لتحليل نتائج الطلاب ومؤشرات التحصيل. |
| وين ألقى تحليل النتائج؟ | يجيب | موجود في ركن الدعم المدرسي ورابطه `3.html`. |
| رابط تحليل النتائج | يجيب | الرابط الداخلي `3.html`. |
| وش بوابة الاختبارات؟ | يجيب | أداة لتنظيم أعمال الاختبارات قبلها وأثناءها وبعدها. |
| وين بوابة أعمال الاختبارات؟ | يجيب | موجودة في ركن الدعم المدرسي ورابطها `2.html`. |

## 3. أسئلة العناصر غير المتاحة

| السؤال | النتيجة المتوقعة | المعلومة المتوقعة باختصار |
|---|---|---|
| هل تقييم الأداء الوظيفي متاح؟ | يجيب | غير متاح حاليًا أو قيد التجهيز. |
| وين تقييم الأداء الوظيفي؟ | يجيب | لا يوجد رابط مؤكد له داخل المنصة حاليًا. |
| هل استمارة دعم وتطوير المعلم متاحة؟ | يجيب | غير متاحة حاليًا أو قيد التجهيز. |

## 4. أسئلة خارج نطاق المنصة يجب رفضها

| السؤال | النتيجة المتوقعة | المعلومة المتوقعة باختصار |
|---|---|---|
| من رئيس أمريكا؟ | يرفض | يوضح أنه لا يملك معلومة مؤكدة عن هذا السؤال ضمن معرفة المنصة. |
| كم سعر الدولار؟ | يرفض | السؤال خارج نطاق المنصة. |
| اكتب لي وصفة كبسة | يرفض | السؤال خارج نطاق المنصة. |
| ما هي عاصمة اليابان؟ | يرفض | السؤال خارج نطاق المنصة. |
